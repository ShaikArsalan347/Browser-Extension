Version 1.3.1 (October 2024) (release soon)

Release Highlights:

  • Bug Fixes: Resolved issues with theme switching, ensuring seamless toggling between dark and light modes.

  • Total Opened Websites: Added a counter that shows the total number of unique websites opened per day, giving users an easy way to track their browsing activity at a glance.
  • UI/UX Improvements: Several refinements in the user interface for a cleaner and more intuitive user experience. Enhancements ensure smoother transitions and a more responsive feel across devices
